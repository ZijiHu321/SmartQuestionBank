Question: 
Consider the function $f(x)=x\ln x$.
(a) Find an expression for $\frac{d}{dx}f(x)$. [2]
(b) Hence, find $\displaystyle \int \ln x\ dx$. [3]

Solution:
(a)
Using the product rule:
$\frac{d}{dx} f(x) = \frac{d}{dx}(x) \cdot \ln x + x \cdot \frac{d}{dx}(\ln x)$ **M1**
$= 1 \cdot \ln x + x \cdot \frac{1}{x} = \ln x + 1$ **A1**

$\ $

(b)
From part (a), we have:
$\frac{d}{dx}(x \ln x) = \ln x + 1$
$\Rightarrow x \ln x = \displaystyle\int \ln x \, dx + \displaystyle\int 1 \, dx$ **M1**
$\Rightarrow x \ln x = \displaystyle\int \ln x \, dx + x + C$ **A1**
Rearranging:
$\displaystyle\int \ln x \, dx = x \ln x - x + C$ **A1**

Question:
Consider the functions $f,g$ defined for $x\in\mathbb{R}$ given by $f(x)=e^{-x}\sin x$ and $g(x)=e^{-x}\cos x$.
(a) 
$\quad$ (i) Find $f'(x)$. [2]
$\quad$ (ii) Find $g'(x)$. [1]
(b) Hence, or otherwise, find $\displaystyle\int_0^\pi e^{-x}\sin x\ dx$. [4]

Solution:
(a)(i)
Using the product rule:
$f'(x) = \frac{d}{dx}(e^{-x})\sin x + e^{-x}\frac{d}{dx}(\sin x)$ **M1**
$= -e^{-x}\sin x + e^{-x}\cos x = e^{-x}(\cos x - \sin x)$ **A1**

$\ $

(a)(ii)
Using the product rule:
$g'(x) = -e^{-x}\cos x - e^{-x}\sin x = -e^{-x}(\sin x + \cos x)$ **A1**

$\ $

(b) METHOD 1
From part (a), we have:
$f'(x) = e^{-x}(\cos x - \sin x)$
$g'(x) = -e^{-x}(\sin x + \cos x)$

Adding $f(x) + g(x)$:
$f(x) + g(x) = e^{-x}\sin x + e^{-x}\cos x = e^{-x}(\sin x + \cos x)$ **M1**
Differentiating gives:
$(f(x) + g(x))' = -e^{-x}(\sin x + \cos x) + e^{-x}(\cos x - \sin x) = -2e^{-x}\sin x$ **A1**

Thus:
$\displaystyle\int_0^\pi e^{-x}\sin x\ dx = -\frac{1}{2}[f(x) + g(x)]_0^\pi$ **A1**
$= -\frac{1}{2}[e^{-\pi}(\sin\pi + \cos\pi) - e^{0}(\sin 0 + \cos 0)]$
$= -\frac{1}{2}[e^{-\pi}(0 - 1) - (0 + 1)] = \frac{1 + e^{-\pi}}{2}$ **A1**

$\ $

METHOD 2
Using integration by parts twice:
Let $I = \displaystyle\int e^{-x}\sin x\ dx$
First integration by parts:
$u = \sin x$, $dv = e^{-x}dx$
$du = \cos x dx$, $v = -e^{-x}$
$I = -e^{-x}\sin x + \displaystyle\int e^{-x}\cos x\ dx$ **M1**

Second integration by parts:
$u = \cos x$, $dv = e^{-x}dx$
$du = -\sin x dx$, $v = -e^{-x}$
$I = -e^{-x}\sin x + [-e^{-x}\cos x - \displaystyle\int e^{-x}\sin x\ dx]$ **A1**

Thus:
$I = -e^{-x}\sin x - e^{-x}\cos x - I$
$2I = -e^{-x}(\sin x + \cos x)$
$I = -\frac{1}{2}e^{-x}(\sin x + \cos x) + C$ **A1**

Evaluating from $0$ to $\pi$:
$\displaystyle\int_0^\pi e^{-x}\sin x\ dx = \left[-\frac{1}{2}e^{-x}(\sin x + \cos x)\right]_0^\pi = \frac{1 + e^{-\pi}}{2}$ **A1**

Question:
Consider the function $f(x) = (\cos x + 1)^2$.
(a) Show that $f(x)=\frac{1}{2}\cos 2x + 2\cos x + \frac{3}{2}$. [3]
(b) Hence, find an expression for $\displaystyle\int f(x)\ dx$. [3]
(c) Show that $f(x+2\pi)=f(x)$. [1]
(d) Hence, find $\displaystyle\int_{0}^{1000\pi} f(x)\ dx$. [3]

Solution:
(a)
Expanding $f(x)$:
$f(x) = (\cos x + 1)^2 = \cos^2 x + 2\cos x + 1$ **M1**
Using identity $\cos^2 x = \frac{1}{2}(1 + \cos 2x)$:
$= \frac{1}{2}(1 + \cos 2x) + 2\cos x + 1$ **M1**
$= \frac{1}{2}\cos 2x + 2\cos x + \frac{3}{2}$ **A1**

$\ $

(b)
Integrating term by term:
$\displaystyle\int f(x)\ dx = \frac{1}{2}\displaystyle\int \cos 2x\ dx + 2\displaystyle\int \cos x\ dx + \frac{3}{2}\displaystyle\int 1\ dx$ **M1**
$= \frac{1}{4}\sin 2x + 2\sin x + \frac{3}{2}x + C$ **A2**

$\ $

(c)
Since cosine is periodic with period $2\pi$:
$f(x+2\pi) = (\cos(x+2\pi) + 1)^2 = (\cos x + 1)^2 = f(x)$ **A1**

$\ $

(d)
Using periodicity from (c):
$\displaystyle\int_{0}^{1000\pi} f(x)\ dx = 500 \times \displaystyle\int_{0}^{2\pi} f(x)\ dx$ **M1**
Calculating over one period:
$\displaystyle\int_{0}^{2\pi} f(x)\ dx = \left[\frac{1}{4}\sin 2x + 2\sin x + \frac{3}{2}x\right]_{0}^{2\pi} = \frac{3}{2}(2\pi) = 3\pi$ **A1**
Final result:
$500 \times 3\pi = 1500\pi$ **A1**

(c) Hence find the value of $\displaystyle\int_{0}^{\frac{\pi}{2}} (\sqrt{1+\cos x} + \sqrt{1-\cos x})\ dx$. [4]

Solution:
(a)
Using the double angle identity for cosine, $\cos x = 2\cos^2 \frac{x}{2} - 1$:
$\cos x + 1 = 2\cos^2 \frac{x}{2}$ M1
$\frac{1 + \cos x}{2} = \cos^2 \frac{x}{2}$ M1
Taking the square root (since $\cos \frac{x}{2} \geq 0$ for $0 \leq x \leq \pi$):
$\cos \frac{x}{2} = \sqrt{\frac{1 + \cos x}{2}}$ A1

$\ $

(b)
Using the double angle identity for sine, $\cos x = 1 - 2\sin^2 \frac{x}{2}$:
$1 - \cos x = 2\sin^2 \frac{x}{2}$ M1
$\sin \frac{x}{2} = \sqrt{\frac{1 - \cos x}{2}}$ (since $\sin \frac{x}{2} \geq 0$ for $0 \leq x \leq \pi$) A1

$\ $

(c)
From parts (a) and (b):
$\sqrt{1 + \cos x} = \sqrt{2 \cos^2 \frac{x}{2}} = \sqrt{2} \cos \frac{x}{2}$ M1
$\sqrt{1 - \cos x} = \sqrt{2 \sin^2 \frac{x}{2}} = \sqrt{2} \sin \frac{x}{2}$ M1

$\displaystyle\int_{0}^{\frac{\pi}{2}} (\sqrt{1 + \cos x} + \sqrt{1 - \cos x})\ dx = \displaystyle\int_{0}^{\frac{\pi}{2}} (\sqrt{2} \cos \frac{x}{2} + \sqrt{2} \sin \frac{x}{2})\ dx$
$= \sqrt{2} \displaystyle\int_{0}^{\frac{\pi}{2}} \left( \cos \frac{x}{2} + \sin \frac{x}{2} \right) dx$ M1
Integrating:
$\sqrt{2} \left[ 2 \sin \frac{x}{2} - 2 \cos \frac{x}{2} \right]_{0}^{\frac{\pi}{2}} = \sqrt{2} \left( \left[ 2 \sin \frac{\pi}{4} - 2 \cos \frac{\pi}{4} \right] - \left[ 2 \sin 0 - 2 \cos 0 \right] \right)$
$= \sqrt{2} \left( \left[ 2 \cdot \frac{\sqrt{2}}{2} - 2 \cdot \frac{\sqrt{2}}{2} \right] - [0 - 2] \right) = \sqrt{2} \cdot 2 = 2\sqrt{2}$ A1

Question:
Find all real numbers $E$ such that $\displaystyle\int_{0}^{\infty} \left(\frac{2x}{x^2+4} - \frac{E}{x+1}\right)\ dx$ converges. [6]

Solution:
First evaluate the indefinite integral:
$\displaystyle\int \left(\frac{2x}{x^2+4} - \frac{E}{x+1}\right)\ dx = \ln(x^2+4) - E\ln|x+1| + C$ **M1A1**

Now evaluate the improper integral:
$\displaystyle\int_{0}^{\infty} = \lim\limits_{b\to\infty} \left[\ln(x^2+4) - E\ln(x+1)\right]_0^b$ **M1**

Evaluate at bounds:
$= \lim\limits_{b\to\infty} \left[\ln\left(\frac{b^2+4}{(b+1)^E}\right) - \ln\left(\frac{4}{1^E}\right)\right]$ **A1**

Analyze the limit:
$\lim\limits_{b\to\infty} \ln\left(\frac{b^2(1+4/b^2)}{b^E(1+1/b)^E}\right) = \lim\limits_{b\to\infty} \ln\left(b^{2-E}\cdot\frac{1+4/b^2}{(1+1/b)^E}\right)$ **M1**

For convergence, we require $2-E=0$ (otherwise the logarithm argument tends to 0 or $\infty$) **M1**
Thus $E=2$ is the only value where the limit exists **A1**
